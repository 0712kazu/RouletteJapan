# 市区町村ルーレット

市区町村の形から名前を当てる、ブラウザで動く地図クイズです。

## 遊び方

1. 出題範囲を「全国」または都道府県から選びます。
2. 「スタート」を押すと市区町村がランダムに選ばれます。
3. 赤いポリゴンの形を見て、5秒以内に市区町村名を考えます。
4. 5秒後に都道府県名と市区町村名が表示されます。

同じ出題範囲を一巡するまでは、同じ自治体を重複して出題しません。

## 起動

ビルドは不要です。ローカルHTTPサーバーから開いてください。

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000/` を開きます。

## 境界データ

国土交通省「国土数値情報（行政区域データ N03）」2026年版を加工しています。

- `N03_004`を市区町村名として使用
- 政令指定都市の行政区は親の市へ結合
- 東京都の特別区は個別に出題
- 行政区域コード単位でポリゴンを結合
- ブラウザ表示用に境界を簡略化
- 都道府県別GeoJSONとして遅延読込

データを再生成するには、GDALの`ogr2ogr`が必要です。

```bash
./scripts/build-municipalities.sh
```

簡略化の強さや処理対象は環境変数で変更できます。

```bash
SIMPLIFY_TOLERANCE=0.001 PREF_CODES="13 14" ./scripts/build-municipalities.sh
```

## 使用技術

- HTML / CSS / Vanilla JavaScript
- Leaflet 1.9.4
- 国土地理院タイル
- 国土数値情報（行政区域データ）

境界データの利用条件は、国土数値情報ダウンロードサイトの最新の案内を確認してください。
